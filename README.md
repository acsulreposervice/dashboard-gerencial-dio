# dashboard-gerencial-dio

## 📋 Sobre o Projeto
O projeto gira em torno do aprendizado das disposições
visuais no contexto dos relatórios e como dispor cada
elemento para uma melhor experiência do usuário.

### 🔍 Desafios Encontrados
Um dos principais desafios foram a questão de configurações
dos visuais. Dependendo da visão, pode conter muitas opções
para alterar. Porém é de ciência que as opções são dominadas
com a devida prática.

É de ciência que, ficou faltando uma página das três requeridas,
mas para o intuito do aprendizado, se mostrou eficiente.

## 🛠️ Tecnologias Utilizada
- **Power BI**: foi utilizado unicamente com seus recursos nativos.
